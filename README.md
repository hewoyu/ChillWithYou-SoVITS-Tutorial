## 全流程操作步骤
### 1. 前置工具与环境准备
1.  下载并安装C++重构版AssetStudio，工具操作可参考对应解包教程与官方说明。
2.  下载并部署GPT-SoVITS-v2pro-20250604一键整合包，部署步骤完全对应核心训练教程。
3.  下载UVR5人声分离工具，用于游戏语音素材的提纯降噪。

### 2. 游戏角色语音素材解包与初筛
1.  打开重构版AssetStudio，通过`File→Load file/Load folder`加载《Chill With You》游戏资源目录，操作完全对应解包教程步骤。
2.  切换至`Asset List`标签页，筛选`Type=AudioClip`类型的资源，定位并导出游戏核心角色的全部原生人声台词。
3.  素材初筛：剔除背景音乐、音效、过短语气音，仅保留清晰的人声干声；最终筛选素材总时长建议3-10分钟，单条音频时长3-10秒，覆盖陈述句、疑问句、感叹句等多种语气，提升模型泛化能力。

### 3. 训练数据集预处理（训练效果核心）
本环节完全对应核心训练教程的数据集制作步骤，是决定音色还原度的关键：
1.  人声提纯：使用UVR5工具对筛选后的游戏语音进行人声分离，彻底去除残留的背景音、混响，仅保留纯净人声干声。
2.  音频切片：使用GPT-SoVITS WebUI内置的「音频切片」工具，将长音频切分为3-10秒的短片段，自动剔除无效片段。
3.  ASR文本标注与校对：使用WebUI内置的「ASR自动标注」功能为切片生成对应文本，**必须手动100%校对每一句文本**，保证文本与音频内容完全一致，避免训练后模型出现吞字、发音不准的问题。

### 4. GPT-SoVITS训练配置
1.  打开GPT-SoVITS-v2pro WebUI，进入「训练」标签页，填写模型名称（仅支持英文/数字，禁止中文、空格、特殊符号）。
2.  按照教程指引，选择训练集路径（预处理完成的音频切片文件夹）、标注文本路径（校对完成的.list标注文件）。
3.  语种选择中文，匹配游戏角色语音环境；v2pro版本必须选择`s2v2Pro.json`专属配置文件，其余参数完全对应核心训练教程的推荐值，可根据显卡显存调整批次大小，避免爆显存。

### 5. 分阶段模型训练
本环节完全对应核心训练教程的分阶段训练逻辑：
1.  **SoVITS模型训练（音色克隆核心）**
    - 确认参数无误后，点击「开启SoVITS训练」，训练轮数参考教程推荐值，游戏短语音素材建议8-10轮，不超过15轮，避免过拟合。
    - 训练完成后，模型会自动保存在`SoVITS_weights`文件夹中。
2.  **GPT模型训练（语义语气适配核心）**
    - 参数设置与SoVITS训练轮数保持一致，点击「开启GPT训练」，等待训练完成。
    - 训练完成后，模型会自动保存在`GPT_weights`文件夹中；本环节训练将让模型适配实时对话的语义与语气，实现自由问答场景下的自然语音生成。

### 6. 模型推理测试与效果验证
1.  进入WebUI「推理」标签页，加载训练完成的SoVITS与GPT模型。
2.  选择训练集内3-10秒的参考音频，填写对应参考文本，选择中文语种。
3.  输入游戏实时对话场景的测试文本，点击「合成语音」，对比训练后语音与游戏原生语音的音色还原度、自然度，验证训练效果。

### 7. API对接与游戏Mod落地
1.  按照核心训练教程的步骤，开启WebUI的API接口，对接《Chill With You》的AI全语音对话Mod。
2.  进入游戏实机测试，验证AI角色使用训练后专属模型完成实时语音问答的效果，完成全流程闭环。

## 常见问题与避坑指南
### 解包环节
1.  重构版AssetStudio仅支持标准Unity/UnityCN资源，若出现资源无法解析，可参考工具原作者视频排查是否为魔改加密资源。
2.  提取的游戏资源仅可用于个人本地训练，严禁二次传播、商用，避免侵权风险。

### 训练环节
1.  必须使用v2pro版本配套的预训练模型，不可使用旧版v1预训练模型，否则会出现训练报错、无法对接Mod的问题。
2.  训练后音色不还原，优先排查数据集：是否为纯净人声干声、文本标注是否100%准确、素材是否覆盖多种语气。
3.  显存不足时，优先降低音频切片长度与训练批次大小，不要强行调高参数，避免爆显存。
4.  训练轮数不可过高，否则会出现过拟合，导致模型仅能还原固定台词，无法适配自由对话场景。

### 对接环节
1.  实时对话出现吞字、重复、语气生硬，优先更换更低轮数的模型，或缩短输入文本长度。
2.  对接Mod前需先在WebUI内完成推理测试，确认模型无问题后再开启API接口，避免对接后出现异常。
